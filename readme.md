# Simple experiment with ECS

https://wolispace.com/eek/

## Features
- ECS architecture
- Basic physics (movement, collision)
- Camera focus
- Zoom
- Frustum culling
- Collision detection
- FPS overlay with live entity count
- Dynamic entity spawning (+/- keys)

## Controls
| Key | Action |
|-----|--------|
| `W A S D` | Move player |
| Scroll wheel | Zoom in / out |
| `+` / `=` | Spawn 100 bouncers |
| `-` | Remove 100 bouncers |
